**1. Add handlers to `gateway/dashboard/settings.py`:**

**`handle_general_tab(request)`** — GET handler for `/settings/general`:
- Deferred imports inside function: `load_config`, `is_managed` from `hermes_cli.config`
- Deferred import: `get_all_toolsets` from `toolsets` (inside function body)
- `config = load_config()`
- Extract these config values with defaults:
  - `memory_enabled = config.get("memory", {}).get("memory_enabled", True)` (bool)
  - `streaming = config.get("display", {}).get("streaming", False)` (bool)
  - `compression_enabled = config.get("compression", {}).get("enabled", True)` (bool)
  - `compression_threshold = config.get("compression", {}).get("threshold", 0.50)` (float)
  - `skin = config.get("display", {}).get("skin", "default")` (string)
  - `current_toolsets = config.get("toolsets", ["hermes-cli"])` (list of strings)
- Get all available toolsets: `all_toolsets = get_all_toolsets()` — build list of dicts with `name` and `enabled` (True if name in current_toolsets)
- Return `aiohttp_jinja2.render_template("settings/_general.html", request, context)` (synchronous, no await)

**`handle_general_save(request)`** — POST handler for `/settings/general/save`:
- Check `is_managed()` — if True, render with `flash="error:Configuration is managed by NixOS/Enclava."`
- `data = await request.post()`
- Inside `async with request.app["config_lock"]:`:
  - `config = load_config()`
  - `config.setdefault("memory", {})["memory_enabled"] = "memory_enabled" in data` (checkbox: present=True, absent=False)
  - `config.setdefault("display", {})["streaming"] = "streaming" in data`
  - `config.setdefault("compression", {})["enabled"] = "compression_enabled" in data`
  - Read `threshold = data.get("compression_threshold", "").strip()` — if non-empty: `config.setdefault("compression", {})["threshold"] = float(threshold)`
  - `config.setdefault("display", {})["skin"] = data.get("skin", "default").strip()`
  - Collect toolsets: `all_toolsets = get_all_toolsets()` then `enabled = [name for name in all_toolsets if data.get(f"toolset_{name}")]` — set `config["toolsets"] = enabled`
  - `save_config(config)`
- Re-render general tab with `flash="success:General settings saved."`

**`handle_status_tab(request)`** — GET handler for `/settings/status`:
- Deferred imports: `load_config` from `hermes_cli.config`
- `config = load_config()`
- `runner = request.app.get("gateway_runner")`
- Build `platforms_status = []`:
  - If `runner` is not None and has `adapters` attribute:
    - For each `platform, adapter` in `runner.adapters.items()`:
      - Append `{"name": platform.value, "connected": adapter.is_connected, "has_error": adapter.has_fatal_error, "error": adapter.fatal_error_message}`
- Build context: `model = config.get("model", "not configured")`, `platforms = platforms_status`, `gateway_running = runner is not None`
- Return `aiohttp_jinja2.render_template("settings/_status.html", request, context)` (no await)
